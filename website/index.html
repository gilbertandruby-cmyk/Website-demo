Quick Installation

1

Clone Repository

git clone https://github.com/gilbertandruby-cmyk/Website-demo.git
cd Website-demo
2

Install Dependencies

npm install
3

Configure Bot

cp .env.example .env
# Edit .env with your settings
4

Start Bot

npm start
